CSS

  • When you create a stylesheet you need to link it in the html using <link rel="stylesheet" href="./location/name.css">
  • Don't forget to reset defaults by having *{ margin:0; padding:0;} as the first line on your stylesheet.
  • To inline style stuff with css place it in the <head> and use <element style="property:value"> or <style> then type like you normally would on a stylesheet and don't forget the closing tag.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element an
  • SPECIFICITY
    ID > CLASSES > ELEMENT
    This is the priority of targeting. !important will overrule other selectors even ones more specific.
  • Targeting
    To target an element just type the name of the element
    To target a class .class-name
    To target an id #id-name
    To target everything use an asterik(*)
    To target specific things chain the class and element by typing both with no comma seperating them
    To target multiple elements, classes, or ids, use commas to seperate each one you target
  • Margin Collapse
    When creating margin with two images or boxes horizontally next to each other they combine the margin to decide the distance from each other.
    When creating margin with two images or boxes vertically next to each other they use the largest margin to decide the distance from each other.
  • Psuedo-Class
    You can use :focus, :visited, :disabled, or :hover following the selector to determine when the CSS initiates. (e.g) h1:hover
