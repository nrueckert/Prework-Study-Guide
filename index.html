HTML

  • The body element represents the visible content shown to the user.
  • Classes can be used multiple times compared to ids only being able to be used once

CSS

  • When you create a stylesheet you need to link it in the html using <link rel="stylesheet" href="./location/name.css">
  • Don't forget to reset defaults by having *{ margin:0; padding:0;} as the first line on your stylesheet.
  • To inline style stuff with css place it in the <head> and use <element style="property:value"> or <style> then type like you normally would on a stylesheet and don't forget the closing tag.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element an
  • SPECIFICITY
    ID > CLASSES > ELEMENT
    This is the priority of targeting. !important will overrule other selectors even ones more specific.
  • Targeting
    To target an element just type the name of the element
    To target a class .class-name
    To target an id #id-name
    To target everything use an asterik(*)
    To target specific things chain the class and element by typing both with no comma seperating them
    To target multiple elements, classes, or ids, use commas to seperate each one you target
  • Margin Collapse
    When creating margin with two images or boxes horizontally next to each other they combine the margin to decide the distance from each other.
    When creating margin with two images or boxes vertically next to each other they use the largest margin to decide the distance from each other.
  • Psuedo-Class
    You can use :focus, :visited, :disabled, or :hover following the selector to determine when the CSS initiates. (e.g) h1:hover

JavaScript

  • use <script src="./location/name.js> after the footer in the <body>
  • JavaScript is read from top to bottom and order counts.
  • if statements work as so: if (condition){
    // block of code to be executed if condition is truthy
    }
  • if else statements work as so: if (first condition) {
    // block of code to be executed if first condition is truthy
    } else if (second condition) {
    // block of code to be executed if second condition is truthy
    } and so on
    Example Given: if (0) {
    console.log("I'm 0!")
    } else if (null) {
    console.log("I'm null!")
    } else if (true) {
    console.log("I'm true!")
    }
  • === is a strict equality operator and checks to see if the two values are equal, and returns a boolean result (true or false)
  • To create a single variable use: var name = value;
  • Arrays hold multiple pieces of data and are used as so: var studentInfo = ["Lu", 54, true]; only 'Lu' has to be in quotation marks because its a string, 54 is a number and true is a boolean so they don't require quotation marks. Arrays are also zero-indexed and sequential meaning the first piece of data is also '0' and the next piece of data '1' and so forth. If you want to access the data in the array you start with the name of the variable that stores the array then add brackets and the index of the data you want. (i.e) console.log[studentInfo[1]; this will display the student's age in the console.
  • a for loop works as so: for(var x = 0; x < array-name.length; x++) {
    console.log(array-name[x]);
    } // This allows the system to automatically change the output for as many as there are by creating a variable 'x' and making the loop go as a long as 'x' is less than the array length. Then inside of the for loop you put whatever code youre wanting it to repeat. Always make your variable start at 0 because arrays start at 0 and then 'variable-name++' at the end to increase its value.
  • /* */ is how you create comments or comment out code.
